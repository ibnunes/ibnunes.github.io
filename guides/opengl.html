General instructions

I recommend performing all operations as root, using su.




1. Install g++

Back to top

g++ is necessary to compile OpenGL projects, so one needs to install it as follows:


dnf install g++

2. Install GLFW

Back to top

I'm using the Downloads directory. Use a directory of your choice. Create a folder glfw where we'll unzip GLFW files downloaded from the official site. 


cd ~/Downloads
mkdir glfw
wget https://github.com/glfw/glfw/releases/download/3.3/glfw-3.3.zip
unzip glfw-3.3.zip

Enter the folder, build and install GLFW:


cd glfw-3.3
dnf builddep glfw
cmake -G "Unix Makefiles"
make
make install
