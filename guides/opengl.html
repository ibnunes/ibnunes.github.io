General instructions

I recommend performing all operations as root, using su.




Install g++

Back to top

g++ is necessary to compile OpenGL projects, so one needs to install it as follows:


dnf install g++

Install GLFW

Back to top

I'm using the Downloads directory. Use a directory of your choice. Create a folder glfw where we'll unzip GLFW files downloaded from the official site. 


cd ~/Downloads
mkdir glfw
wget https://github.com/glfw/glfw/releases/download/3.3/glfw-3.3.zip
unzip glfw-3.3.zip

Enter the folder, build and install GLFW:


cd glfw-3.3
dnf builddep glfw
cmake -G "Unix Makefiles"
make
make install

Automate things

Back to top

Lets automate things a little bit with bash or zsh.

  1. Create a folder to save the scripts: 
    
# For BASH
mkdir ~/.bashrc.d

# For ZSH
mkdir ~/.zshrc.d
  2. Append the following code to ~/.bashrc or ~/.zshrc 
    
# For BASH
for cfg in "$HOME"/.bahrc.d/*.bash; do

# For ZSH
for cfg in "$HOME"/.zshrc.d/*.zsh; do

# For BOTH, add the following after the previous line
    . "$cfg"
done
unset -v cfg
  3. OPTIONAL to compile C files: Create the file gcc.bash or gcc.zsh in the directory we previously created with this script.
  4. Create the file gl.bash or gl.zsh in the directory we previously created with this script.
  5. Reload the resource script for bash or zsh with source.
