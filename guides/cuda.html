Add repositories

Back to top

First of all, we need to install the necessary repositories from negativo17:


dnf config-manager --add-repo=https://negativo17.org/repos/fedora-nvidia.repo

Install CUDA

Back to top

Since we already have the NVidia graphic drivers installed using this guide, all we need now is the following:


dnf install cuda nvidia-driver-cuda cuda-devel
